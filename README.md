Live Example: https://example-hono-bun.forgeon.io/

Prerequisites:

- [Forgeon CLI](https://forgeon.io/docs/cli) installed globally

To develop locally:

```
bun install
forge dev
```

```
open http://localhost:3000
```

To build locally:

```
bun install
forge build
```

To deploy:

```
bun install
forge deploy
```
